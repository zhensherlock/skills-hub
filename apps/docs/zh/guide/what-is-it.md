# 什么是 Skills Hub？

**Skills Hub** 是一个企业级技能中心，基于企查查 MCP 提供企业调查与分析工具集。

这些 Skills 可以帮助用户快速完成企业背调、尽职调查、风险评估等专业任务，通过一键安装即可使用。

## 主要特性

- **企业调查**: 快速企业风险排查、背景调查，全面了解目标企业。
- **尽职调查**: 并购与投资尽职调查，专业级尽调报告生成。
- **求职背调**: 猎头职位验证、求职公司调查，避免入职风险。
- **资信评估**: 客户信用评估、供应商准入审查，降低交易风险。
- **一键安装**: 通过 URL 一键下载安装 Skill，快速上手使用。
- **企查查数据**: 基于企查查官方数据源，信息准确可靠。

### headhunter-verify 🎯

猎头职位验证 - 验证猎头推荐职位真实性、核实企业背景。

### job-company-check 💼

求职公司调查 - 面试前了解目标公司、评估入职风险。

### enterprise-risk-check 🔍

企业快速风险排查 - 快速了解企业风险状况、合作可行性判断。

### mma-dd 🤝

并购尽职调查 - 并购目标全面尽调、股权收购分析。

### investment-dd 💰

投资尽职调查 - 被投企业尽调、估值分析、投资风险评估。

### customer-credit 💳

客户资信评估 - 客户信用评估、赊销审查、账期审批。

### supplier-audit 📋

供应商准入审查 - 供应商资质核查、履约能力评估。
