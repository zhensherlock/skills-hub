# What is Skills Hub?

**Skills Hub** is an enterprise-level skill center that provides enterprise investigation and analysis tools based on QCC (企查查) MCP.

These Skills help users quickly complete professional tasks such as enterprise background checks, due diligence, and risk assessment. They can be installed with one click and used immediately.

## Key Features

- **Enterprise Investigation**: Quick enterprise risk assessment and background checks for comprehensive company insights.
- **Due Diligence**: M&A and investment due diligence with professional-grade report generation.
- **Job Background Check**: Headhunter position verification and company investigation to avoid employment risks.
- **Credit Assessment**: Customer credit evaluation and supplier admission review to reduce transaction risks.
- **One-Click Install**: One-click download and install Skills via URL, get started quickly.
- **QCC Data Source**: Based on official QCC (企查查) data source, accurate and reliable information.
