# Getting Started

## How to Use

In OpenClaw, simply enter the following command to install and use a Skill:

```
https://raw.githubusercontent.com/zhensherlock/skills-hub/main/releases/enterprise-risk-check-v1.0.0.zip Download the zip package and unzip it, help me install this skill, then set the environment variable QCC_AGENT_API_KEY=xxxx
```

> **Note**: Replace `xxxx` in `QCC_AGENT_API_KEY=xxxx` with your QCC API Key.

## Get API Key

1. Visit https://agent.qcc.com to log in or register
2. Get your personal API Key from the console
3. Enter the API Key into the environment variable in the command above
